Fileless Malware Simulation

Built and tested malware that runs only in memory

Developed malware that executed entirely in memory without leaving traces on disk. Set up to restart automatically with system-level priviledges and simulated real attack behaviors like capturing screenshots and communicating with a command server. Applied hiding techniques including encryption and misleading file names to avoid detection. This project showed how challenging it is to defend against memory-only threats and why security teams need behavior-based monitoring, not just signature scanning.

  • Python (server & encryption)
  • PowerShell scripting
  • Windows Task Scheduler
  • Data encryption & obfuscation

KRACK Attack Simulation

Demonstrated a real Wi-Fi security flaw that lets attackers intercept encrypted traffic.

Built a controlled test network to exploit a critical weakness in WPA2 Wi-Fi security. By manipulating the connection handshake, I forced devices to reuse encryption keys, which allowed me to capture and decrypt supposedly secure traffic. Used Wireshark to verify the attack worked and confirmed that unpatched devices exposed their data. This demonstrated why keeping network devices updated is crucial for security.

  • Wi-Fi security testing
  • Network traffic analysis
  • Wireshark
  • Vulnerability research

Mindwell

Developed an AI system that detects cognitive distortion patterns in text to advance mental health research.

MindWell is a research-driven project aimed at enhancing mental health support through artificial intelligence. Conducted as my B.Sc. Computer Science capstone project, it focused on building a comprehensive dataset and AI models to detect and classify 15 types of cognitive distortions (unhealthy thinking patterns) in written text. I contributed to data collection from social media and AI-generated text, annotation design, and model evaluation, while co-developing a framework to make AI decisions more explainable. Our models achieved ~76% accuracy, significantly outperforming baselines (36–40%) and demonstrating potential for ethical AI in mental health contexts. The project culminated in a technical research paper documenting our dataset, methodology, and results.

  • Python . PyTorch · AI/ML frameworks
  • Natural language processing
  • Data collection & annotation
  • Model design & evaluation
  • Research methodology
